# New Ubuntu Install

Ubuntu 20.04.3 LTS

## Settings

- Appearance > Dark
- Displays > Night Light > ON
- Mouse & Touchpad > Touchpad > Natural Scrolling > OFF
- Power > When the Power Button is pressed
- Nautilus > Hamburger Menu > Show Hidden Files
- Additional Drivers > Install proprietary drivers
